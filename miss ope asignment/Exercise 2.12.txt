Given that y = ax2 + 5x + 2, which of the following are correct Java statements for this equation?
 a) y = a * x * x + 5 * x + 2; 
 b) y = a * x * x + (5 * x) + 2; 
 c) y = a * x * x + 5 * (x + 2); 
 d) y = a * (x * x) + 5 * x + 2; 
 e) y = a * x * (x + 5 * x) + 2;
 f) y = a * (x * x + 5 * x + 2); 
        Solution
 a)y = a * x * x + 5 * x + 2;
